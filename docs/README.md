# JellyfishBot — 平台使用指南

欢迎使用 JellyfishBot！本文帮助你了解平台能做什么、以及如何让 Agent 帮你完成工作。

---

## 平台功能

### 对话

- 与 AI Agent 进行多轮对话，支持流式输出
- 管理多个对话，自动生成标题
- Agent 会记住每个对话的完整上下文，切换回之前的对话时能继续
- 支持语音输入（长按录音，自动转文字）
- 支持图片上传与识别

### 文件系统

- 每个用户拥有独立的虚拟文件系统，通过右上角按钮打开文件面板
- 支持浏览、上传、下载、编辑、重命名、删除、拖拽上传
- 文件分为三个主要目录：
  - `/docs/` — 文档目录，存放参考资料和输出文件
  - `/scripts/` — 脚本目录，存放可执行的 Python 脚本
  - `/generated/` — Agent 生成的内容

### 设置中心

在左上角的齿轮图标进入设置，包含以下页面：

| 页面 | 功能 |
|------|------|
| **System Prompt** | 编辑 Agent 的系统提示词，支持版本管理与回滚 |
| **User Profile** | 设置用户画像，Agent 会据此个性化回答 |
| **Subagent** | 配置子 Agent（默认内置 deep-research 和 memory） |
| **Python 环境** | 管理可用的 Python 包 |
| **批量运行** | 上传 Excel，Agent 逐行处理并输出结果（入口：**设置 → 通用**） |
| **Service 管理** | 发布 AI 服务供外部调用（微信、API） |
| **定时任务** | 查看和管理 Agent 创建的定时任务 |
| **微信接入** | 将 Service 接入微信 |
| **收件箱** | 接收来自 Service 用户的消息和请求 |
| **通用** | 时区设置、界面样式（Regular / Terminal）切换 |

### 主题与样式

- **明暗模式**：点击左下角 JellyfishBot 旁的太阳/月亮图标切换
- **界面样式**：在 设置 > 通用 中选择 Regular（默认）或 Terminal（等宽字体、直角、CRT 扫描线效果）

## Agent 能力

你可以通过对话让 Agent 帮你完成以下事情：

### 文件与脚本

- **读写文件** — 创建、编辑、搜索你的文档
- **执行 Python 脚本** — Agent 可以编写并运行 Python 代码，支持 numpy、pandas、matplotlib、plotly 等数据科学库
- **展示媒体** — 生成的图表、音频、视频、PDF 可以直接在对话中展示

### 信息检索

- **网络搜索** — Agent 可以搜索实时信息
- **网页抓取** — Agent 可以获取并解析指定网页内容
- **深度研究** — 内置 deep-research 子 Agent，可进行多步骤的深入调研

### 计划与审批

- **任务计划** — Agent 可以提出分步计划，你审批后逐步执行
- **人机协作** — 关键操作可配置为需要你确认后才执行

### 定时任务

- 告诉 Agent 你需要定期执行的事情（如"每天早上 9 点汇总新闻"）
- Agent 会自动创建 cron / 一次性 / 周期性 定时任务
- 可通过微信等渠道推送执行结果

### AI 生成

在对话头部的能力开关中启用后，Agent 还可以：
- **生成图片**（DALL·E）
- **生成语音**（TTS）
- **生成视频**

### 记忆

- **memory 子 Agent** — Agent 可以回溯你的历史对话、检索跨对话信息
- **Soul 编辑**（可选）— Agent 可以自主更新系统 Prompt 和用户画像来优化自身表现

## 定制 Agent 行为

### System Prompt

在 设置 > System Prompt 中编辑 Agent 的核心行为指令。可用占位符：

| 占位符 | 说明 |
|--------|------|
| `{today}` | 自动替换为当前日期 |
| `{time}` | 自动替换为当前时间和时区 |
| `{user_profile_context}` | 自动插入用户画像信息 |

每次保存会自动创建版本快照，可随时回滚。

### User Profile

在 设置 > System Prompt 中编辑用户画像，告诉 Agent 你的背景和偏好，Agent 会据此个性化回答。

## Service（对外服务）

你可以将 Agent 能力打包为 Service 供外部用户使用：

1. 在 设置 > Service 管理 中创建 Service
2. 选择 Prompt 版本、可访问的文档、允许的能力
3. 生成 API Key 供外部调用
4. 可选接入微信，让微信用户直接与你的 Agent 对话

Service 用户只能访问你指定的文档，权限是隔离的。

## 常见问题

**Q：Agent 能记住之前说过什么吗？**
能。每个对话的上下文会完整保存，切换回去时 Agent 能继续之前的话题。memory 子 Agent 还可以跨对话检索你的历史信息。

**Q：脚本执行有什么限制？**
脚本在独立沙箱中执行，有超时限制。可用的 Python 包可以在 设置 > Python 环境 中查看和安装。

**Q：如何让 Agent 定期执行任务？**
直接在对话中告诉 Agent，如"每周一早上帮我生成周报"。Agent 会自动创建定时任务，你可以在 设置 > 定时任务 中管理。

**Q：如何批量处理数据？**
在 **设置 → 通用** 页面的「批量运行」区块中上传 Excel 文件，配置好 Agent 要对每行做什么，即可自动逐行处理。
